# Tech Stack

## Frontend

- **React**: Biblioteca JavaScript para construir interfaces de usuario.
- **Vite**: Herramienta de construcción rápida para proyectos web.
- **TypeScript**: Superset de JavaScript que añade tipado estático.
- **Tailwind CSS**: Framework CSS para un desarrollo rápido de UI.

## Backend

- **Node.js**: Entorno de ejecución JavaScript del lado del servidor.
- **Express**: Framework web para Node.js.

## Servicios de IA

- **Azure OpenAI Service**: Para generar recomendaciones de IA.

## Despliegue

- **Vercel**: Plataforma para el despliegue de aplicaciones web.

## Consideraciones Arquitectónicas

- **Monorepo**: El proyecto está estructurado como un monorepo con el frontend y el backend en directorios separados (`src/` para el frontend y `server/` para el backend).
- **Comunicación Cliente-Servidor**: La comunicación se realiza a través de APIs REST.
