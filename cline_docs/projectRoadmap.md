# Project Roadmap

## Overarching Goals and Key Features

- [ ] Desarrollar una aplicación web para la gestión de riesgos y posiciones laborales.
- [ ] Integrar servicios de IA para recomendaciones.
- [x] Desplegar la aplicación en Vercel. (Despliegue inicial en progreso)

## Completion Criteria and Progress Tracking

- [ ] Interfaz de usuario para formularios de empresa y posición laboral.
- [ ] Interfaz de usuario para visualización de documentos de riesgo y recomendaciones de IA.
- [ ] Conexión con el backend para el procesamiento de datos.
- [ ] Despliegue exitoso en Vercel.

## Completed Tasks

- Creación de la estructura inicial del proyecto.
- Implementación de componentes de UI (`CompanyForm`, `JobPositionForm`, `RiskDocument`, `AIRecommendation`).
- Configuración del servicio Azure OpenAI.
- Configuración del servidor backend.

## Future Scalability Considerations

- Implementación de autenticación y autorización de usuarios.
- Base de datos persistente para almacenar información de empresas, posiciones y riesgos.
- Mejoras en la interfaz de usuario y experiencia del usuario.
- Integración con otros servicios de IA o APIs externas.
